# OpenRAM Setup Guide

This repository provides a step-by-step guide for installing OpenRAM, configuring dependencies, setting up the SKY130 PDK, and generating SRAM macros.

## Overview

OpenRAM is an open-source memory compiler that generates SRAM macros for various technologies, including SKY130. This guide is intended for students, researchers, and engineers who want to:

- Install OpenRAM
- Configure required dependencies
- Set up the SKY130 technology files
- Generate SRAM macros
- Run example designs and simulations

- Installing system dependencies
- Installing Python and creating a virtual environment
- Installing Nix (optional)
- Installing OpenRAM
- Configuring environment variables
- Setting up the SKY130 PDK
- Running a sample SRAM macro
